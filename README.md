# FruitShoppingMall

## :calendar: 진행 기간 
- 2022년 11월 ~ 2022년 12월
 
## :page_facing_up: 프로젝트 목적
- JSP를 활용한 MVC 패턴 및 CRUD 숙련도 향상을 위함

## :page_facing_up: 프로젝트 내용
- 컵과일 및 과일 도시락 판매 서비스

## :mag: 기술 스택
- `JSP`, `서블릿`, `MVC(프론트컨트롤러)`, `MariaDB`, `JDBC`, `HTML`, `JavaScript`, `CSS`
